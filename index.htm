La fuite de l'espace


 

Le jeu se déroule dans un vaisseau spatiale, l'objectif du jeu est de faire des tâches pour sortir du vaisseau spatiale le plus vite possible car il y a une fuite d'oxygène ce qui causera la mort des passagers. Toutes les portes du vaisseau sont verrouillées. Pour les déverrouillées tu devras accomplir les tâches suivantes : à chaque fois que tu réussi à ouvrir une porte un chiffre apparait sur l’écran. Ce chiffre va appartenir à un code que tu devras saisir sur la porte finale pour sortir de cet vaisseau spatiale avec la navette de secours. Tu as 50 minutes pour y arriver, passé ce délai tu mourras asphyxié.


 

Les joueurs doivent résoudre quatre énigmes. Après avoir résoud un énigme, ils receveront un chiffre qui appartient a un code qui leurs permettera de déverouiller la porte de fuite.
